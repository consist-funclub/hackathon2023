# ハッカソンとは？

ハッカソン（Hackathon）とは、ハック（Hack）とマラソン（Marathon）を組み合わせた造語。短期間でアイディアを出し開発と作品発表を行うコンテストです。<br>
コンシスト社内ハッカソンでは4名程度のチームで業務システムの開発を行います。
